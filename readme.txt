== Description ==

Traffic Link is a great tool to promote your blog to other similar blogs. Our unique algorithm automatically finds similar blogs with yours and it shows your link on them enabling you to reach users that are interested in your niche.
<br/>
The widget is very easy to use, just install the plugin and put the widget in your sidebar, from there on it will show relevant links and your site will also appear on other sites that use our plugin.

== Installation ==

If you are using FTP, upload the Traffic Link plugin to your blog(in the wp-content/plugins directory) (be sure to upload the "TrafficLink" directory or it  will not work).

Activate it in the plugins menu in your wp-admin control panel and voila! you're done!

You can also upload using the wordpress administrator upload page under Plugins > Add New (and then Upload at the top menu inside the plugins page)
